# Hub Frontend

React + Vite PWA for FS-Remote Hub.

## Features

- **Liquid Glass Theme** - Modern glassmorphism design
- **Responsive Layouts** - Portrait (top/bottom) and landscape (side-by-side)
- **PWA Support** - Add to Home Screen
- **Tool Integration** - Dev Remote, VS Code, Xcode

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## PWA

The app is a Progressive Web App (PWA) that can be installed on iPhone via Safari's "Add to Home Screen" feature.
